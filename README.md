Тестовое задание Playwright (Python)

Описание задачи
Необходимо написать автотест с использованием Playwright (Python), который:

- Открывает веб-страницу https://playwright.dev/
- Проверяет соответствие заголовка страницы ожидаемому значению
- Запускается минимум в двух браузерах

Используемые технологии
- Python
- Playwright
- Pytest

Запуск проекта

1. Установить браузеры Playwright:
   playwright install

2. Запустить тест:
   pytest task.py

Ожидаемый результат
Тест успешно проходит проверку заголовка страницы в следующих браузерах:
- Chrom
- Firefox
